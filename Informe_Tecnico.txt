Informe Técnico – Calculadora Web

1. Nombre del Proyecto: Calculadora Web Básica
2. Tecnologías Usadas:
- HTML, CSS, JavaScript
- Node.js + Jest (para pruebas)
- Git (control de versiones)
- Docker (contenedorización)

3. Funcionalidad:
La aplicación permite realizar operaciones matemáticas básicas (suma, resta, multiplicación y división) a través de una interfaz web sencilla.

4. Pruebas Unitarias:
Se implementaron pruebas con Jest para verificar el correcto funcionamiento del cálculo matemático. Se probaron casos válidos e inválidos.

5. Control de Versiones:
Se utilizaron dos ramas principales: main (estable) y develop (desarrollo). Los cambios se integran a main después de ser validados en develop.

6. Docker:
Se creó un Dockerfile para automatizar la instalación de dependencias y la ejecución de pruebas.

7. Conclusión:
Este proyecto cumple con los requisitos básicos de una aplicación web funcional y modular, ideal para extenderse con nuevas funciones como historial o almacenamiento local.
